Dev and design projects

DataRips - Custom datasheet editor

Frontend web developer and UI designer

Custom datasheet editor for DataRips clients (health workers) to upload their patients data. This way, DataRips clients have a custom tool to manage their monthly datasheets and DataRips has an organised way to get normalized data to process it later.

Tools used:

  • React.js
  • Firebase
  • Redux
  • Figma.

It does not use any datasheet editor library, it was created and customized from the bottom.

Archedi: A historical data visualization

Full stack web developer and UX designer

Data visualization to show historical facts in a spiral timeline and a bubble map. Along with the Vivelab Bogotá and the Universidad Nacional de Colombia, we created an interesting way for people to approach to historical data and get involved in the process of making history. I led the development process and created the design of both the mobile and the pc experience.

Tools used:

  • Angular.js
  • Firebase
  • D3.js
  • Mapbox API
  • AdobeXD

Historical files - database admin

Fullstack web developer and UI designer

Admin for Universidad Nacional de Colombia History department personel, to fill a history files database with different formats (video, images, text, newspaper, audio, etc). The challenge here was creating a form that applied to all the different kinds of files and the characterization data each one should have. This way, all the data can be usable for analysis and visualizations like timelines and maps.

Tools used:

  • React.js
  • Firebase
  • Semantic UI

CIPADE: Formalization guide for productive initiatives

Fullstack web developer and UX designer

This project in progress is a personal guide for productive initiatives in Colombia that want to get formalized but don't know how. It consists on a series of tasks divided by phases that the project team must complete one by one, following several formats and assistants provided by the platform. This web application was sponsored by the Trust for the Americas, UNAL, OEA among others.

Tools used:

  • React.js
  • Frebase
  • Redux
  • Adobe XD
